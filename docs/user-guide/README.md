## 功能概览

### AI 智能对话
- 多会话管理，按时间分组
- RAG 知识库检索
- MCP 工具扩展
- 多模型并行对比

### 技能系统
- 预配置 AI 行为模式（导师模式、制卡模式等）
- 按需加载工具组（知识检索、笔记、Anki 等）
- 三级加载：内置 → 全局 → 项目级
- 用户自定义技能（SKILL.md 格式）

### 学习资源中心
- 访达风格文件管理
- 笔记（Markdown 编辑器）
- 教材（PDF 阅读器）
- 题目集（AI 识别）
- 翻译、作文批改
- 知识导图
- 向量化索引管理

### Anki 智能制卡
- 多格式文档解析
- AI 智能分段和生成
- 3D 卡片预览
- APKG 导出 / AnkiConnect 同步
- 多模板支持

### 数据管理
- 分层备份策略
- 自动备份
- WebDAV / S3 云同步
- 数据治理面板
