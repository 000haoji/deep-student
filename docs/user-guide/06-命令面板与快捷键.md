# 命令面板与快捷键

## 概述

命令面板是 Deep Student 的效率中心，通过键盘即可触达应用的所有功能。

**快捷键**：
- **macOS**: `Cmd + K`
- **Windows/Linux**: `Ctrl + K`

---

## 常用命令

### 全局导航
| 命令 | 说明 |
|------|------|
| `跳转到智能对话` | 进入 Chat V2 主界面 |
| `跳转到学习资源` | 打开文件管理器 (VFS) |
| `跳转到 Anki 制卡` | 进入制卡工作台 |
| `跳转到技能管理` | 管理 Skills |
| `跳转到设置` | 打开系统设置 |

### 对话操作
| 命令 | 说明 |
|------|------|
| `新建对话` | 创建一个新的聊天会话 |
| `切换模型` | 快速切换当前对话使用的 AI 模型 |
| `激活技能` | 搜索并激活特定技能 (如 `/skill`) |
| `开启/关闭推理模式` | 切换深度思考模式 |

### 资源管理
| 命令 | 说明 |
|------|------|
| `新建笔记` | 创建一篇新的 Markdown 笔记 |
| `导入教材` | 导入 PDF/文档到库中 |
| `全局搜索` | 搜索所有笔记、题目和聊天记录 |

---

## 自定义快捷键

你可以为常用命令绑定个性化快捷键：

1. 打开 **设置 -> 快捷键**。
2. 找到目标命令。
3. 点击编辑按钮，按下你想要的组合键。
