# 学习资源中心

学习资源中心（Learning Hub）是 Deep Student 的核心功能模块，提供类似访达（Finder）风格的文件管理体验，让你可以集中管理所有学习资源，包括笔记、教材、题目集、翻译、作文和知识导图等。

## 概述

学习资源中心采用分栏布局设计：
- **左侧快捷导航栏**：快速访问不同类型的资源和系统功能
- **中间文件列表区**：显示当前目录下的所有文件和文件夹
- **右侧应用面板**：点击资源后在此区域打开对应的查看/编辑器

### 支持的资源类型

| 资源类型 | 说明 |
|---------|------|
| 笔记 | Markdown 格式的学习笔记 |
| 教材 | PDF、Word、Excel、PPT 等文档 |
| 题目集 | 通过 AI 识别的题目及答案 |
| 翻译 | 中英文翻译练习 |
| 作文 | 作文批改和评分 |
| 知识导图 | 思维导图形式的知识整理 |
| 图片 | 学习相关的图片文件 |
| 文档 | 其他格式的文档文件 |

## 快速开始

### 进入学习资源中心

1. 在应用侧边栏点击「学习资源」图标
2. 系统将显示学习资源中心的全屏界面

### 创建第一个笔记

1. 在快捷导航栏上方，点击 **+** 按钮
2. 在弹出菜单中选择「新建笔记」
3. 系统自动创建一个新笔记并在右侧面板中打开编辑器
4. 开始编写你的学习笔记

### 导入教材

1. 点击 **+** 按钮，选择「导入教材」
2. 在文件选择对话框中选择要导入的文档（支持 PDF、Word、Excel、PPT 等格式）
3. 等待导入完成，系统会自动进行 OCR 识别和向量化索引
4. 导入完成后可在教材查看器中阅读

## 功能详解

### 1. 快捷导航栏

快捷导航栏位于界面左侧，分为多个分组：

#### 快捷访问
- **桌面**：显示用户添加的桌面快捷方式
- **全部文件**：显示所有资源（按文件夹结构）
- **最近**：显示最近访问过的资源
- **收藏**：显示已标记为收藏的资源

#### 资源类型
- **全部笔记**：筛选显示所有笔记
- **全部教材**：筛选显示所有教材
- **全部题目集**：筛选显示所有题目集
- **全部作文**：筛选显示所有作文批改
- **全部翻译**：筛选显示所有翻译练习
- **知识导图**：筛选显示所有思维导图

#### 媒体
- **全部图片**：筛选显示所有图片文件
- **全部文档**：筛选显示其他文档文件

#### 系统
- **回收站**：查看已删除的资源（可恢复或永久删除）
- **向量化状态**：查看资源的 AI 索引状态
- **记忆管理**：管理 AI 对话中的长期记忆

### 2. 搜索功能

在快捷导航栏顶部有搜索框，支持：
- 按资源名称搜索
- 按内容搜索（需要资源已完成向量化索引）
- 搜索结果实时显示

**使用方法**：
1. 点击搜索框或按快捷键
2. 输入搜索关键词
3. 系统会在 300ms 后自动搜索（防抖处理）
4. 点击搜索结果即可打开对应资源

### 3. 新建资源

点击快捷导航栏顶部的 **+** 按钮，可以新建：

| 操作 | 说明 |
|------|------|
| 新建文件夹 | 创建新的文件夹用于组织资源 |
| 新建笔记 | 创建空白 Markdown 笔记 |
| 新建题目集 | 创建空白题目集，可拍照识别题目 |
| 导入教材 | 选择本地文档导入为教材 |
| 新建翻译 | 创建空白翻译练习 |
| 新建作文 | 创建空白作文批改任务 |
| 新建知识导图 | 创建空白思维导图 |

### 4. 文件列表

文件列表区支持两种视图模式：

#### 列表视图
- 显示资源图标、名称、修改时间、文件大小
- 适合快速浏览大量文件
- 悬停时显示更多操作按钮

#### 网格视图
- 以卡片形式显示资源
- 显示较大的预览图标
- 适合视觉化浏览

**切换视图**：在底部工具栏点击视图切换按钮

### 5. 文件操作

#### 打开资源
- **双击**资源可在右侧应用面板中打开
- 点击资源会选中该资源

#### 大文件预览提示
- 文档类资源（DOCX/XLSX/PPTX/文本等）超过 10MB 时会提示“文件过大，无法预览”
- PDF 会显示“大文件加载”提示并继续尝试加载（流式）
- PDF 预览支持应用内预览与系统打开；当仅有 Base64 且过大时会提示无法预览

#### 选择资源
- **单击**：选中单个资源
- **Ctrl/Cmd + 点击**：切换选中状态（多选）
- **Shift + 点击**：范围选择
- **Ctrl/Cmd + A**：全选当前目录下的所有资源
- **框选**：在网格视图中拖动鼠标框选多个资源

#### 移动资源
- **拖拽**：将资源拖拽到目标文件夹上即可移动
- 支持多选拖拽移动
- 拖拽时会显示移动的资源数量

### 6. 右键菜单

在不同位置右键点击会显示不同的菜单：

#### 空白区域右键
- 新建文件夹
- 新建笔记
- 新建题目集
- 导入教材
- 新建翻译
- 新建作文
- 新建知识导图
- 刷新

#### 文件夹右键
- 打开
- 新建子文件夹
- 在此新建笔记
- 重命名
- 删除

#### 资源右键
- 打开
- 重命名
- 收藏 / 取消收藏
- 添加到桌面
- 删除

### 7. 批量操作

选中多个资源后，底部工具栏会显示批量操作按钮：

| 操作 | 说明 |
|------|------|
| 全选 | 选中当前目录下所有资源 |
| 取消选择 | 清除所有选中状态 |
| 添加到对话 | 将选中资源批量添加到 AI 对话上下文 |
| 移动 | 将选中资源移动到其他文件夹 |
| 删除 | 将选中资源移动到回收站 |

**快捷键**：
- `Ctrl/Cmd + A`：全选
- `Delete` 或 `Backspace`：删除选中项
- `Escape`：取消选择

### 8. 收藏功能

对于重要的资源，可以添加到收藏：
1. 右键点击资源，选择「收藏」
2. 收藏的资源会显示星标图标
3. 在快捷导航栏点击「收藏」可快速访问所有收藏的资源
4. 再次右键可「取消收藏」

### 9. 回收站

删除的资源不会立即永久删除，而是移动到回收站：

#### 查看回收站
点击快捷导航栏中的「回收站」

#### 恢复资源
右键点击回收站中的资源，选择「恢复」

#### 永久删除
右键点击资源，选择「永久删除」

#### 清空回收站
- 右键点击回收站空白区域，选择「清空回收站」
- 或点击移动端工具栏中的清空按钮
- **注意**：此操作不可撤销

### 10. 向量化状态

向量化是让 AI 能够理解和搜索资源内容的关键步骤。

#### 查看索引状态
点击快捷导航栏中的「向量化状态」

#### 索引状态说明
| 状态 | 说明 |
|------|------|
| 已索引 | 资源已完成向量化，可用于 AI 搜索 |
| 待索引 | 资源等待索引处理 |
| 索引中 | 资源正在进行索引 |
| 失败 | 索引失败，可点击重试 |
| 不适用 | 该资源类型不支持索引 |

#### 手动索引
- 点击「一键索引」按钮批量处理所有待索引资源
- 点击单个资源旁的刷新按钮可单独重新索引

#### 召回测试
在向量化状态页面可以进行召回测试，验证索引效果。

### 11. 记忆管理

记忆管理用于存储 AI 对话中的长期记忆。

> **提示**：配合 `deep-scholar` (深度学者) 技能，AI 可以更智能地调用和组织这些长期记忆，辅助复杂的学术研究。

#### 首次设置
1. 点击「记忆管理」
2. 选择或创建一个文件夹作为记忆根目录
3. 系统会将记忆以笔记形式存储在该文件夹中

#### 管理记忆
- 搜索记忆
- 创建新记忆
- 编辑现有记忆
- 删除不需要的记忆

### 12. 桌面视图

桌面视图提供快捷方式功能，方便快速访问常用资源。

#### 添加快捷方式
1. 在文件列表中右键点击资源
2. 选择「添加到桌面」
3. 该资源的快捷方式会出现在桌面视图中

#### 管理快捷方式
- 点击快捷方式打开对应资源
- 右键点击可重命名或移除快捷方式

#### 设置桌面根目录
桌面可以关联一个文件夹作为根目录，在桌面新建的资源会自动保存到该目录。

## 移动端适配

在移动设备上，学习资源中心采用三屏滑动布局：

### 三屏布局
1. **左侧**：应用入口（快捷创建资源）
2. **中间**：文件视图（文件列表）
3. **右侧**：应用内容（打开的资源编辑器）

### 手势操作
- 左右滑动切换三个屏幕
- 点击资源自动切换到右侧应用视图
- 点击顶部返回箭头返回文件视图

### 移动端工具栏
移动端顶部有简化的工具栏：
- 搜索按钮
- 新建文件夹按钮
- 新建笔记按钮

## 常见问题

### Q: 如何将资源添加到 AI 对话？
**A**: 选中一个或多个资源后，点击底部工具栏的「添加到对话」按钮，即可将资源添加到当前 AI 对话的上下文中。

### Q: 导入教材时显示"导入中"很久不动？
**A**: 大型 PDF 文件需要进行 OCR 识别和预览渲染，可能需要较长时间。进度条会显示当前处理的页数。

### Q: 为什么有些资源显示"向量化失败"？
**A**: 可能的原因：
- 资源内容为空
- 文件格式不支持
- 网络连接问题（需要 AI 服务）

可以在「向量化状态」页面点击重试按钮重新索引。

### Q: 删除的资源可以恢复吗？
**A**: 是的，删除的资源会先移动到回收站，可以在回收站中恢复。只有「永久删除」或「清空回收站」后才会彻底删除。

### Q: 如何快速找到某个资源？
**A**: 
1. 使用搜索功能（输入资源名称或内容关键词）
2. 使用资源类型筛选（如"全部笔记"）
3. 使用收藏功能标记常用资源
4. 使用桌面快捷方式

### Q: 支持哪些教材格式？
**A**: 支持以下格式：
- PDF
- Word 文档（.docx）
- Excel 表格（.xlsx、.xls、.xlsb、.ods、.csv）
- PowerPoint 演示文稿（.pptx）
- 电子书（.epub）
- 文本文件（.txt、.md、.html）
- 数据文件（.json、.xml）

**注意**：旧版 Word 格式（.doc）暂不支持。
