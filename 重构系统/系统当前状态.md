## 快速启动

### Windows用户
```batch
# 方式1：使用批处理文件
start_all.bat

# 方式2：手动启动
# 终端1 - 启动后端
cd backend
python app_local.py

# 终端2 - 启动前端
cd frontend
npm run dev
```

### 访问地址
- 前端界面：http://localhost:3001
- API文档：http://localhost:8000/docs
